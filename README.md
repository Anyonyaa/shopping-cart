# Mini E-commerce Shopping Cart 🛒

## Description
A dynamic shopping cart app simulating a real e-commerce experience. Users can add/remove products, adjust quantities, and see subtotal, tax, and total in real-time. Cart persists across page refreshes using localStorage, demonstrating frontend state management and responsive UI.

## Features
- Add products to cart  
- Increase/decrease quantity without removing items  
- Remove items from cart  
- Real-time calculation of subtotal, tax (10%), and total  
- Cart persists using localStorage  
- Responsive design for desktop and mobile  

## Technologies
- HTML  
- CSS  
- JavaScript

## How to Run
1. Clone the repository  
2. Open `index.html` in a browser  
3. Start adding products to see the cart in action
